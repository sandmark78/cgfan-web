## Prompt

请生成一张「旅途窗口叙事海报」。

【可替换内容】
交通工具：【飞机 / 火车 / 汽车 / 轮渡 / 巴士】
窗外场景：【云海日出 / 荒漠峡谷 / 雨夜城市 / 午夜港口】
主题情绪：【启程 / 探索 / 夜归 / 告别】
主标题：【填写英文标题】
辅助短句：【填写一句简短文案】
主色调：【冷灰蓝 / 暖金橙 / 雨夜蓝红 / 深海蓝金】
画面比例：9:16

采用真实乘客视角，从座位一侧斜着看向窗外，窗口不要正对镜头，也不要完整居中。窗框、座椅、扶手或桌角自然进入画面，形成真实的空间遮挡和主观观察感。

交通工具内部保持昏暗，窗外景色明亮，形成明显的内暗外亮关系。玻璃带有轻微反光、水汽、雨滴、划痕或车厢灯光倒影，让画面像旅途中偶然看到的一瞬间。

强化移动状态：靠近窗口的树木、灌木、电线杆、路牌、灯光或水面出现明显的方向性动态模糊，远处山体、城市、云层或灯塔相对清晰，形成"近景有速度、远景有故事"的层次。

整体使用电影感旅行摄影、复古杂志编辑风、自然胶片颗粒、真实景深和轻微暗角。画面具有情绪、速度和旅途感，不要像静态风景照，也不要像规整的旅游广告模板。

文字必须成为画面的一部分：
- 根据主题选择高对比衬线体、窄长字体或手写字体
- 主标题可以局部进入窗框、阴影、雨滴或玻璃反射
- 允许轻微拖影、雾化、反光、磨损或水面延伸字效
- 文字要有设计感，但保持克制，不使用夸张3D、金属或炫技特效
- 小字统一使用简洁窄体无衬线字体
- 保留少量编辑元素，如 ROUTE JOURNAL、ISSUE 编号、坐标、细线或地点标签

画面构图不必绝对工整，允许自然倾斜、局部裁切和不对称留白。真实摄影质感，高级、沉浸、有电影叙事感，避免正面窗口、完整对称窗框、静止景色、过度锐化、过饱和、模板化排版和文字堆积。

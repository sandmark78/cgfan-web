## Prompt

input:
product_function = [any product function]
inspiration_source = [any biological / mechanical / cultural / natural source]
user_context = [optional]
tech_features = [optional]
style_level = [minimal / premium / playful / luxury / futuristic / ai-infers]

run biomorphic_product_extraction:

product_dna := infer product category, usage scenario, user interaction, power system, controls, safety needs, and ergonomic constraints
inspiration_dna := infer silhouette, anatomy, texture, motion, structural logic, surface rhythm, protective forms, sensory cues, or symbolic traits
function_mapping := map inspiration traits to product functions:
  - fins → airflow / cooling / acoustic dispersion if relevant
  - shell → protection / enclosure if relevant
  - beak / snout / head → sensor / output / light / nozzle if relevant
  - wings → folding / circulation / coverage if relevant
  - scales / plates → vents / grip / modular panels if relevant
  - roots / veins → cable routes / fluid paths / support if relevant
  - eyes / spots → indicators / sensors / leds if relevant

material_system := infer ceramic, aluminum, fabric, recycled plastic, glass, rubber, wood, resin, carbon, silicone, or other appropriate materials
cmf := infer color, material, finish, texture, surface softness, gloss, transparency, and accent colors
product_name := generate a concise brandable name derived from function + inspiration
feature_list := generate realistic product-specific features
detail_panels := infer 4 close-up views that explain the most important features
layout := infer hero render + spec card + detail grid presentation

render:
a premium biomorphic product concept sheet with a hero product render and industrial-design feature documentation.

forbidden:
hard-coded dinosaur, hard-coded lamp, hard-coded speaker, hard-coded purifier, hard-coded fan, copied names, random anatomy, fake features unrelated to product function.

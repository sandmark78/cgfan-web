# 设计质量检查清单

**使用时机**：每次大改后、上线前、用户反馈"感觉不对"时

## 一致性

- [ ] 所有圆角值统一（卡片 16px? 22px? 全站一致？）
- [ ] 所有阴影风格统一（方向、颜色、层数）
- [ ] 所有间距遵循 4px 网格
- [ ] 所有 hover 效果风格统一
- [ ] 所有字体大小在 4 级层级内
- [ ] 亮色/暗色模式都检查过

## 层次

- [ ] 页面有且只有一个视觉焦点
- [ ] 信息层级清晰（标题 > 副标题 > 正文 > 辅助文字）
- [ ] 卡片和背景有明确的层次分离
- [ ] 导航和内容不抢注意力

## 呼吸感

- [ ] 留白足够（不挤）
- [ ] 段落之间有呼吸
- [ ] 卡片之间有间距
- [ ] 页面边缘有边距

## 色彩

- [ ] 主色不超过 3 个
- [ ] 阴影带环境色
- [ ] 亮色/暗色模式色彩和谐
- [ ] 文字对比度达标（WCAG AA 4.5:1）
- [ ] 颜色总数不超过 5 种

## 玻璃效果

- [ ] 背景能透出来
- [ ] 边缘有可见的棱边光
- [ ] 有方向性高光
- [ ] blur 值合理（不过度）
- [ ] saturate 让背后颜色鲜活
- [ ] 透明度符合要求（暗色 42-60%，浅色 35-48%）

## 移动端

- [ ] 375px 宽度下布局正常
- [ ] 触控目标 ≥ 44px
- [ ] 文字 ≥ 14px
- [ ] 无横向溢出
- [ ] 搜索和导航可用
- [ ] 间距 = 桌面端 × 0.75

## 动效

- [ ] 有过渡（不突兀）
- [ ] 不过度（不花哨）
- [ ] 缓动曲线统一（cubic-bezier(0.22, 1, 0.36, 1)）
- [ ] 暗色模式动效幅度适当
- [ ] hover 效果克制（轻微上浮 + 阴影加深）

## 排版

- [ ] 行高符合要求（正文 1.6-1.8，标题 1.2-1.3）
- [ ] 字号层级不超过 4 级
- [ ] 中文正文 ≥ 15px，英文 ≥ 14px
- [ ] 对齐方式统一
- [ ] 字体不超过 2 种

## 禁忌检查

- [ ] 无纯黑阴影
- [ ] 无白叠白
- [ ] 无超过 3 层 backdrop-filter
- [ ] 无所有元素都加 hover
- [ ] 无圆角不统一
- [ ] 无暖奶油色调
- [ ] 无过度模糊（blur > 24px）
- [ ] 无不透明暗色卡片（opacity > 70%）
- [ ] 无渐变色按钮
- [ ] 无粗边框（> 2px）
